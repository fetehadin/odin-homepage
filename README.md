# Odin Homepage

A responsive portfolio-style homepage built as part of the **Advanced HTML & CSS** course from **The Odin Project**.

The goal of this project is to practice building responsive layouts from a design specification while applying modern HTML and CSS techniques. This implementation uses **Tailwind CSS** to recreate the provided designs across desktop, tablet, and mobile screen sizes.

## Features

- Fully responsive layout
- Portfolio-style landing page
- Projects showcase section
- Contact section
- Modern UI built with Tailwind CSS
- Mobile-first responsive design

## Built With

- HTML5
- Tailwind CSS
- JavaScript
- Vite
